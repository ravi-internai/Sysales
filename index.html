🔌 Amazon SP-API Setup

1

Node.js Install Karein

Download: nodejs.org — LTS version

2

Backend Setup

cd specialyou-backend
npm install
copy .env.example .env

3

Amazon Credentials

sellercentral.amazon.in → Apps & Services → Develop Apps → Add New App Client

4

.env File Fill Karein

LWA_CLIENT_ID=amzn1.xxx
LWA_CLIENT_SECRET=xxx
LWA_REFRESH_TOKEN=Atzr|xxx
SELLER_ID=Axxx

5

Server Start Karein

node server.js
